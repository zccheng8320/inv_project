# 進銷貨系統
本專案將以前DOS倚天中文系統的進銷貨管理系統利用ASP.NET MVC的全部重新撰寫。

進銷貨系統的主要功能為以下:

1.客戶、廠商...等的資料管理。

2.進貨、銷貨、收款的管理。

### 使用工具
ASP.NET MVC 5
### 資料庫:
MS SQL Server
### 前端:
JQuery、Ajax、Bootstrap、Bootstrap Table

### 資料庫轉移
先將DOS倚天中文系統的dbf檔轉檔為csv檔，並寫程式匯入資料庫中。
